# Chikuwa Teams Mod

# 説明

本プログラムは、`Microsoft Teams` で利用できる `Greasmonkey` 用のMODです。
元々の目的は `チクチク言葉` を `フワフワ言葉` へ変換するのを目的として作成していました。
使うためには、個人でビルドが必要です。
